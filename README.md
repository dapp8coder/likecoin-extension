# Liker Coin 啪啪啪

## 安装

1. 进入extension界面
  * Chrome: chrome://extensions
  * Brave: brave://extensions
1. 打开Developer Mode
  ![Developer Mode](img/developer-mode.png)
1. 点击Load unpacked
  ![Select Unpacked Code](img/select-src.png)
1. 浏览器显示导入extension成功
  ![Success Loaded](img/select-src.png)
